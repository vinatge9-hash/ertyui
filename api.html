API Documentation

Below are placeholder code samples to illustrate how to call the VerifyNow API. Replace endpoints with your production URL and keys.

Example: Verify an email via OTP (curl)

curl -X POST https://api.verifynow.example/v1/verify \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"email": "user@example.com", "otp": "123456"}'

Example: Verify an email using Fetch (JavaScript)

fetch('https://api.verifynow.example/v1/verify', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_API_KEY'
  },
  body: JSON.stringify({ email: 'user@example.com', otp: '123456' })
})
  .then(res => res.json())
  .then(data => console.log(data));
